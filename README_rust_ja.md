# Rust 言語機能デモ

Rust の進化を、各エディションや主要バージョンごとのデモプロジェクトで解説します。

## プロジェクト一覧

| プロジェクト | エディション / 特徴 | 主な学習ポイント |
| :--- | :--- | :--- |
| [rust1.0](./rust1.0/) | Rust 1.0 (The Beginning) | 所有権・借用、パターンマッチ、トレイト |
| [rust2018](./rust2018/) | Edition 2018 | 非同期処理 (`async`/`.await`)、NLL、モジュールシステム刷新 |
| [rust2021](./rust2021/) | Edition 2021 | クロージャの分離キャプチャ、Prelude の拡張、Cargo resolver v2 |

## 学習の進め方

1. **rust1.0**: まずは Rust の根幹であるメモリ安全性の仕組み（所有権）を理解します。
2. **rust2018**: 生産性を劇的に向上させた非同期構文や、借用チェッカの緩和（NLL）を学びます。
3. **rust2021**: より洗練された言語仕様と、エコシステムの改善を確認します。

各ディレクトリの `README_ja.md` では、機能の定義だけでなく、その機能によって**「エンジニアのどのような苦痛が解消されたか」**に焦点を当てて解説しています。

## 共通の実行方法

各プロジェクトディレクトリにて：

```bash
make build    # ビルド
make run      # 実行
make test     # テスト実行
```
